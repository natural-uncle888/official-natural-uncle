使用說明與 HTML 編輯教學
Summary / Bullets 可切換「純文字」或「HTML」模式
1) 基本流程
  1. 點 載入伺服器：讀取目前網站上的 /news.json
  2. 左側選擇公告 → 右側編輯內容
  3. 最下方會顯示「前台真實樣式」預覽
  4. 完成後：下載 news.json（備份） 或 一鍵上傳 GitHub（更新網站）
2) 純文字 vs HTML 模式差異
  • 純文字：你輸入的內容會被當作文字顯示（不會解析 HTML 標籤）。
  • HTML：允許使用 <b> / <span style=""> 等標籤做排版與上色。
  • 建議：一般敘述用「純文字」，需要顏色/放大/置中再切到「HTML」。
3) 你可以用的 HTML 範例
常見文字
<b>粗體</b>
<i>斜體</i>
<u>底線</u>
<br> ← 換行
顏色 / 字體大小 / 置中
<span style="color:#e60000">紅色提醒</span>
<span style="font-size:18px">18px 放大</span>
<span style="text-align:center;display:block">置中內容</span>

提示：如果你要在 bullets 內做「多行」，可用 <br>，或把內容拆成多行（每行一段 bullet）。

4) 安全提醒
  • 請不要貼入未知來源的 HTML/JS（例如 <script>），避免安全風險。
  • GitHub Token 建議使用 Fine-grained token，只授權單一 repo 的 Contents 權限。
